# canvas-dial

[Live demo](http://demo.thi.ng/umbrella/canvas-dial/)

Customizable canvas-based radial dial component with mouse & touch
support.

```bash
git clone https://github.com/thi-ng/umbrella.git
cd umbrella/examples/canvas-dial
yarn install
yarn start
```
