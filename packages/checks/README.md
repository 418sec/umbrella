## About

This package provides ~50 single-function sub-modules for type, feature
& value checks.

## Installation

```bash
yarn add @thi.ng/checks
```

## Dependencies

None

## Usage

```ts
// full import
import * as checks from "@thi.ng/checks";

// individual import
import { isFunction } from "@thi.ng/checks/is-function";
```
