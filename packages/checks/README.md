# @thi.ng/checks

Single-function sub-modules for type, feature & value checks.

## Installation

```
yarn add @thi.ng/checks
```

## Usage

```js
// full import
import * as checks from "@thi.ng/checks";

// individual import
import { isFunction } from "@thi.ng/checks/is-function";
```
